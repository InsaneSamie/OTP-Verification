# Email OTP Verification

This project demonstrates a simple email OTP (One Time Password) verification system using HTML, JavaScript, and SMTP.js. The user enters their email address, receives an OTP via email, and then enters the OTP to verify their email address.

https://github.com/InsaneSamie/OTP-Verification/assets/101932418/5c776d94-e04c-406e-a63c-248d734bda8e

## Table of Contents
- [Features](#features)
- [Installation](#installation)

## Features
- Generate a random OTP and send it to the user's email.
- Verify the OTP entered by the user.
- Simple and intuitive user interface.

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/email-otp-verification.git
    ```
2. Navigate to the project directory:
    ```bash
    cd email-otp-verification
    ```

## Usage
1. Open `index.html` in your web browser.
2. Enter your email address and click the "Send OTP" button.
3. Check your email for the OTP.
4. Enter the OTP in the input field and click the "Verify" button.

## Files
- `index.html`: Contains the structure of the web page.
- `index.css`: Contains the styles for the web page.
- `emailotp.js`: Contains the JavaScript logic for sending and verifying the OTP.
